POST /api/annotation/:id

Paramètres Obligatoires Caractérisent les annotations.
id L'id de la vidéo.
nom Le nom de l'annotation.
commentaire Le commentaire de l'annotation.
timecodeDebut Le timecode de début de l'annotation.
timecodeFin Le timecode de fin de l'annotation.
Paramètres Optionnels Caractérisent les quadrants.
x Le point de coordonnées x du quadrant.
y Le point de coordonnées y du quadrant.
Volonte La valeur de l'attribut volonté.
Imagination La valeur de l'attribut imagination.
Perception La valeur de l'attribut perception.
Entrainement La valeur de l'attribut entrainement.
Memoire La valeur de l'attribut mémoire.
