REST API

Videos

GET /api/video Liste l'ensemble des informations des vidéos présentes sur le serveur.
GET /api/video/:nameVideo Permet de lancer l'interface web de streaming vidéo.
GET /api/video/:id/annotations Liste l'ensemble des annotations d'une vidéo donnée.
GET /api/video/:id/observations Liste l'ensemble des observations d'une vidéo donnée.
GET /api/video/quadrant/:id Obtenir le quadrant d'une vidéo donnée.
POST /api/video Référencer une vidéo sur le serveur.
POST /api/video/:id Associer à une vidéo des données binaires.
DELETE /api/video/:id Supprimer une vidéo donnée.

Annotations

GET /api/annotation Lister toutes les annotations du serveur.
POST /api/annotation/:id Poster une annotation sur une vidéo donnée.
DELETE /api/annotation/:id Supprimer une annotation donnée.

Observations

GET /api/observation Lister toutes les observations du serveur.
POST /api/observation/:id Poster une observation sur une vidéo donnée.
